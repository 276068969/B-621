# 架构文档 (Architecture)

## 1. 技术栈
- **后端**: PHP 8.2 (Apache 驱动)
- **数据库**: MySQL 8.0 (通过 PDO 连接)
- **前端**: HTML5, CSS3, JavaScript (Bootstrap 5.3, TinyMCE 6)
- **容器化**: Docker & Docker Compose

## 2. 系统结构
本系统采用典型的 **LAMP/WAMP** 架构，分为三层：
- **表现层**: 基于 Bootstrap 5 的响应式前端界面，支持 PC 与移动端。
- **业务逻辑层**: 纯 PHP 脚本处理路由、权限校验与业务逻辑。
- **数据持久层**: MySQL 数据库，存储用户、帖子、评论等核心数据。

## 3. 核心模块
- **用户系统**: 包含注册、登录、权限管理（普通用户与管理员）。
- **帖子系统**: 包含发布、展示、搜索与分类。
- **评论系统**: 支持帖子下方的多级展示。
- **后台管理**: 独立的管理界面，支持对帖子与评论的软删除。

## 4. 目录结构
- `/public`: Web 根目录，包含所有入口文件。
- `/includes`: 核心逻辑、数据库连接、视图模板 (layout.php) 与工具函数。
- `/docker`: 容器配置（Apache、初始化脚本）。
- `/docs`: 项目文档。
