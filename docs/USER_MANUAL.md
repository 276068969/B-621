# 用户手册 (User Manual)

## 1. 快速入门
1. 访问首页，点击右侧“立即登录”。
2. 使用预设账号 `demo` (密码 `123456`) 进行体验。
3. 登录后点击“发布新帖”即可开始创作。

## 2. 功能说明

### 前台功能
- **帖子浏览**: 在首页查看所有已发布的帖子，点击进入详情。
- **发布新帖**: 支持富文本编辑（加粗、列表、链接等）。
- **评论互动**: 在帖子下方发表评论。

### 后台功能
- **数据统计**: 查看总用户数、今日新增等核心指标。
- **内容审计**: 管理员可对不当言论进行“删除”（软删除，不影响数据库索引）。
- **快速切换**: 通过右侧“预览前台”随时检查前台展示效果。

## 3. 常见问题
- **Q: 为什么我不能删除帖子？**
  - A: 只有管理员权限才能进入后台执行删除操作。
- **Q: 忘记密码怎么办？**
  - A: 演示项目暂未对接短信/邮件系统，请联系系统管理员手动重置数据库。
