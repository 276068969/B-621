# 实施计划 (Implementation Plan)

## 第一阶段：需求对齐与架构优化
- 梳理用户 8 项核心需求（UI、汉化、Modal、导航、Logo等）。
- 修复 Docker 构建瓶颈，切换阿里云源提升国内构建速度。
- 确立以 `#2c3e50` 为核心的视觉系统。

## 第二阶段：UI/UX 深度重构
- **导航栏**: 实现前后台独立 Topbar，增加 Logo 与高亮下划线。
- **卡片系统**: 统一圆角与阴影，增加 Hover 上浮微交互。
- **表单优化**: 引入 Floating Labels，统一错误提示位置。

## 第三阶段：交互升级与功能补全
- **Modal 替换**: 全局封装 `showModal` 与 `showConfirmModal`，替换所有原生弹窗。
- **汉化补全**: 修复漏掉的页脚文案、登录提示与表单校验提示。
- **后台增强**: 重新设计后台概览页卡片，增加 SVG 图标。

## 第四阶段：验证与交付
- 跨设备兼容性测试。
- 编写全套技术与用户文档。
